# 🛡️ Guia de Cibersegurança para Micro e Pequenas Empresas  
**Cybersecurity Guide for Small and Micro Businesses**

Este repositório reúne modelos, políticas, procedimentos e documentação essencial para ajudar micro e pequenas empresas a implementar práticas básicas de cibersegurança de forma simples, acessível e prática.  
This repository provides templates, policies, procedures, and essential documentation to help small and micro businesses implement basic cybersecurity practices in a simple, accessible, and practical way.
